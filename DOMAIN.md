# Security Domain Knowledge

## The Problem At Scale
- 11,000 alerts per day per enterprise SOC
- 3.5 million unfilled cybersecurity roles globally
- Mean time to detect breach: 204 days
- Average breach cost: $4.9 million
- Analyst turnover: 40%+ annually (alert fatigue)

## The Three Users I Am Building For
1. Tier 1 Analyst: needs speed + clarity
2. Tier 2 Analyst: needs depth + context  
3. CISO: needs narrative + business framing
Same data. Three different products.

## Tools My Users Already Use
- SIEM: Splunk, Microsoft Sentinel, Google Chronicle
- EDR: CrowdStrike, Microsoft Defender, SentinelOne
- SOAR: Palo Alto XSOAR, Splunk SOAR
- Threat Intel: Recorded Future, Mandiant (Google)

## The 5 AI Security Use Cases
1. Alert triage and prioritisation ← I am building this
2. Behavioural anomaly detection
3. Automated incident response (agentic)
4. AI-driven vulnerability prioritisation
5. Natural language interfaces for security data

## My Product Thesis
Alert fatigue is a solvable problem.
The solution is not more rules.
It is AI that understands context, learns
baseline behaviour, and surfaces signal
from noise — with humans staying in the
loop for high-stakes decisions.
